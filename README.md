# Briantest Provider for CloudIaC

## 尝试使用
- 在CloudIaC SaaS版本中添加云模板
- 指定工作目录为 example
- 使用云模板部署环境即可看到结果

## 自己编译
- 克隆仓库到本地
- 进入仓库目录
- 编辑 build.sh 中 EMAIL 变量的值
- 执行 build.sh
